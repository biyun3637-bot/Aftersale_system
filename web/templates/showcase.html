支持 LLM 与关键词混合分类，精准识别退款、订单异常、售后等多种客户意图。

多通道数据查询

Shopify API + 17Track 物流追踪，接口查不到时自动触发 RPA 浏览器抓取兜底。

规则决策引擎

根据金额、物流状态、风险等级综合判定：自动全额退款 / 部分退款 / 转人工审核。

自动化退款执行

对接 Shopify Admin API 自动执行退款，人工审核批准后同样触发退款流程。
